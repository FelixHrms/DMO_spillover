
• $data contains all the work done on data:
  - $data/input is the folder where we save the raw data. 
  - $data_build is divided in:
	- $do_files where we save the scripts to create the datasets
	- $datasets where we save all the datasets used in the analysis
  - $data/analysis is where we save every script to make analyses (regressions, tables, graphs...)
	- The subfolders are for different kind of analysis (descriptive analysis, event study, LP, ...)
  - $data/output is where we save every output of the analysis. 
  	- The subfolders are either for tables and graphs. 
  - $data/output/results is where we put paper-like material 
• $literature is for every paper/document we read about the project.
• $notes is where we save some notes we write down throughout the project (e.g., summary of what we have done, some anecdotal evidence, etc.).
